# 완주군봉동읍법정리의관할구역경계변경에관한조례

##### 제1조 (관할구역 편입)

① 완주군 봉동읍 용암리의 관할구역에 완주군 봉동읍 구암리·구미리·둔산리의 "별표 1"지역을 편입한다. 
② 완주군 봉동읍 낙평리의 관할구역에  완주군 봉동읍  신성리의 "별표 2"지역을 편입한다. <신설 1999.12.15> 
③ 완주군 봉동읍 둔산리의 관할구역에 완주군 봉동읍 용암리·장구리·구암리의 "별표 3"지역을 편입한다. <신설 2001.10.18> 
④ 완주군 봉동읍 용암리의 관할 구역에 완주군 봉동읍 둔산리의 "별표 4"지역을 편입한다. <신설 2001.10.18>

##### 제2조 (관할구역 제외)

① 완주군 봉동읍 구암리·구미리·둔산리의  관할구역중 [별표]지역을 봉동읍 구암리·구미리·둔산리의 관할구역에서 제외한다. 
② 완주군 봉동읍 신성리의 관할구역중 "별표 2" 지역을  완주군 봉동읍 신성리의 관할구역에서 제외한다. <신설 1999.12.15> 
③ 완주군 봉동읍 용암리 ·장구리 ·구암리의 관할구역중 "별표 3"지역을 완주군 봉동읍 용암리·장구리·구암리의 관할구역에서 제외한다. <신설 2001.10.18> 
④ 완주군 봉동읍 둔산리의 관할구역중 "별표 4"지역을 완주군 봉동읍 둔산리의 관할구역에서 제외한다. <신설 2001.10.18>
